# Furigana Tooltip

Chrome extension that shows furigana (phonetic hiragana) in a hover tooltip when you select Japanese kanji text — **zero page modification, zero HTML injection**.

## How it works

Select any Japanese text on any webpage → a tooltip appears above your selection showing the kanji with furigana above them → tooltip disappears when you click elsewhere or deselect.

Runs entirely offline using [kuromoji.js](https://github.com/takuyaa/kuromoji.js) for morphological analysis.

## Install

```bash
npm install        # installs kuromoji and copies vendor files automatically
```

Then in Chrome:

1. Go to `chrome://extensions`
2. Enable **Developer mode** (top right)
3. Click **Load unpacked**
4. Select this directory

The extension is now active on all pages.

## Usage

Highlight any Japanese text. The tooltip appears instantly with furigana above every kanji token. Click anywhere or press Escape to dismiss.

- Works on static HTML, dynamic React/Vue apps, SPAs — anything
- Supports mouse selection and keyboard selection (Shift+Arrow)
- Selections over 150 characters are ignored (keeps the tooltip readable)
- First selection after page load triggers a ~1s dict load; everything after is instant

## Project layout

```
manifest.json        Chrome extension manifest (MV3)
content_script.js    Selection detection + tooltip rendering
tooltip.css          Tooltip styles (all: initial isolation)
scripts/
  copy-vendor.js     Copies kuromoji browser build + dict files to vendor/
vendor/              Generated by npm install — not tracked in git
  kuromoji.js
  dict/
    *.dat.gz
```
